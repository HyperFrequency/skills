# Context7 Documentation Lookup

Source: https://mcpmarket.com/tools/skills/context7-documentation-lookup

## Use When

- You need to find API documentation for libraries (NautilusTrader, pandas, numpy).
- Looking up technical guides or implementation examples.
- Researching library features and best practices.

## Inputs To Ask For

- Library or framework name.
- Specific function, class, or module to look up.
- Version constraints if applicable.

## Outputs

- Relevant documentation excerpts.
- Code examples and usage patterns.
- Links to official documentation sources.

## Workflow

1. Identify the library and specific topic.
2. Search documentation sources using Context7.
3. Extract relevant sections and examples.
4. Provide clear, actionable documentation summary.
5. Include code snippets where applicable.
